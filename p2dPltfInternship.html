Internship

I had a summer internship at Permadeath Studio. I worked on two (yet to be released) projects. I was allowed to talk about my work on the last project.

Project

It’s a die and retry 2d platformer featuring multiplayer and a level editor available for consumers.

Rendering pixel-art

The game world is based on tilesets, I had to find a system where those graphics were to be efficiently rendered.

I made a pixel perfect, mesh based, tile layer renderer.

With that were somme useful optimisation additions, like single drawcall rendering, screen based occlusion culling and tileset batching.

Platformer controller

The gameplay was inspired by games like Super Meat Boy and Bloody Trapland.

I had to program a controller that would fit the designer’s expectations.

After that we had the artist make the various animations and graphic effects that I implemented within the game.

Online multiplayer

I made an online mode prototype where one player could host a game and another could join it by entering the host’s IP address.

Events of the game are to be correctly called on both host and client side.

Also, I set-up prediction in many parts of the game to smooth out the experience.

Level editor

I talked with the team about the constraints of public level editor and how it has to impact the art direction. Once my feedback was taken in, I started making the level editor.

In the end I was able to allow an easy edition of level walls, traps and object within the game engine itself.
