A tile based 3D terrain editor.
Platform

Unity Editor

Tool used

Unity3D

Context


I got inspired by Ocean Horn's terrain graphics and wanted to find out how they achieved it.

How it works.

The terrain editor workds based on a height map. It convert it to a three dimentional voxel map before drawing the apropriate mesh at each voxel's position.

Artist friendly.

The editor uses meshes as base tile for the final terrain's construction.

Meaning different type of terrain can easily be made in either Maya or 3ds Max by modeling each "tile" before importing it into Unity.

I made a tri-planar maping shader so it's fast and easy to texture. Textur map can be big enough to break the feeling of repetition.
