A 2D side-scroller physics engine.
Language

C++

Tool used

CodeBlock
SDL
Tiled

Context


I wanted to program a game inspired by Castlevania, to do so I had to discover the making of a 2D physics engine.

How it works.

Maps are created trough the Tiled Map Editor, then exported as XML files.

The engine loads polygons and lines stored into the file and put them into RAM.

It then uses those static shapes for raycasting as object moves in the level, allowing for fast movements.
